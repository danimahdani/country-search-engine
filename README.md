First, Instal node_modules:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Tech Stack

- Next JS 13 App Dir
- Typescript
- Tailwind CSS
- SVGR
- axios
- react-icons

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
